## Jupyter Notebook cheatsheet
- shift enter: run cell, select below
- ctrl enter: run cell
- alt enter: run cell, insert below
- b: new cell
- x: delete cell
- z: undo delete cell
- y: code cell
- m: markdown cell
- h: view shortcuts
